# Логические значения

Числовые и строковые типы данных имеют большое или бесконечное количество возможных значений. Логический тип данных, напротив, имеет только два допустимых логических значения, представленных литералами true и false. Логическое значение говорит об истинности чего-то, т.е. является ли это что-то истинным или нет.
